i-gard

Manage Easier

i-gard is an Artificial Intelligence-based tool that functions to make farmers more efficient in inspecting and managing
plantations regularly by shortening the distance, time, and energy needed by farmers.

Features

Realtime Detection

Users will get information on temperature, humidity, and livecam
to check the garden which can be accessed through the i-gard application.

Advantage

Bot Responsive

The bot will send a message such as information or a warning via chat box when the watering time has been come
based on the detected parameters. Users only need to press the "yes" or "no" button in response to what the tool will do.

Organize

Watering Schedule

Users can set their own watering schedule or choose the recommended watering
time provided by the system based on the plant data that has been filled in.

 

Value

i-gard can't only be used in one type of plantation but can be used in all of types plantations.
